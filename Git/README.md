## Git 學習大綱
### 1. Git 基礎概念
- Git 簡介與版本控制概念
- Git 的安裝與設定
- Git 儲存庫（Repository）概念
### 2. 基本操作
- git init：初始化儲存庫
- git clone：複製遠端儲存庫
- git add：加入暫存區
- git commit：提交變更
- git status：檢查狀態
### 3. 分支管理
- git branch：建立與管理分支
- git checkout/switch：切換分支
- git merge：合併分支
- 解決合併衝突
4. 遠端操作
- git remote：管理遠端儲存庫
- git push：推送到遠端
- git pull：從遠端取得更新
- git fetch：擷取遠端更新
### 5. 進階技巧
- git rebase：重整分支
- git cherry-pick：挑選提交
- git stash：暫存工作區
- git tag：版本標籤管理
