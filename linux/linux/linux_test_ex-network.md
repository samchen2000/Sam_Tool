## linux命令測試是否有外網
1. ping命令：可以通過ping命令測試是否能夠與外網進行通信。打開終端，輸入以下命令： \
“`
ping http://www.baidu.com 
“` \
如果能夠成功收到來自外網的回復，則表示可以訪問外網；如果顯示”Destination Host Unreachable”或者無法收到回復，則表示無法訪問外網。
2. traceroute命令：可以通過traceroute命令查看到達指定網站所經過的路由路徑，從而判斷是否能夠到達目標網站。打開終端，輸入以下命令： \
“`
traceroute http://www.baidu.com
“` \
如果顯示了經過的路由器IP位址，則表示可以到達目標網站；如果顯示”* * *”，則表示無法到達目標網站。
3. curl命令：可以通過curl命令來測試訪問指定網站是否成功。打開終端，輸入以下命令： \
“`
curl http://www.baidu.com
“` \
如果成功返回了網站的內容，則表示可以訪問外網；如果無法返回網站的內容，或者顯示了錯誤資訊，則表示無法訪問外網。
4. nslookup命令：通過nslookup命令可以查詢指定功能變數名稱的IP位址，從而判斷系統是否能夠解析功能變數名稱。打開終端，輸入以下命令： \
“`
nslookup http://www.baidu.com
“` \
如果成功返回了解析到的IP位址，則表示可以解析功能變數名稱；如果無法返回IP位址，或者顯示了錯誤資訊，則表示無法解析功能變數名稱。
